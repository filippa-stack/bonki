Sacred Session — Dubbelt skydd

LAYER 1 — Header (UI Prevention)
"Nytt kapitel"-knapp disabled + helper text
"Vårt utrymme"-länk gömd
Condition: !!normalizedSession.sessionId
LAYER 2 — ActiveSessionGuard (Route Enforcement)
Redirect till /card/:activeId på alla rutter utom / och /card/:activeId
Condition: appMode && cardId (från NormalizedSessionContext)
LAYER 3 — RelationSettings (Feature Lock)
"Skapa nytt utrymme"-knapp disabled + helper text
Condition: appMode === 'SESSION_ACTIVE' || 'SESSION_WAITING' && !!cardId

Notifikationssystem

IN-APP (usePartnerNotifications)
prompt_notes INSERT (shared) → toast "Din partner delade en tanke"
couple_sessions INSERT (active, partner) → toast om notifyConversationProgress=true
Dedupliciering: 10s window, ID-baserad dismiss
REALTIME BANNERS (system_events)
partner_joined → PartnerConnectedBanner
partner_left_space → PartnerLeftBanner
new_space_created → NewChapterBanner ← NY
Alla: check on mount + realtime subscribe + localStorage one-time gate
E-POST (send-notification-email edge function)
Kräver: notify_email_proposal=true i notification_preferences
Händelser: nytt förslag, delad reflektion
Dedupliciering: 10min via system_events
