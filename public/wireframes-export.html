0A · Provider-stack (React Context, ytterst → innerst)

Aktiva providers i appens träd

QueryClientProvider
TanStack Query · global cache · yttersta lagret
SiteSettingsProvider
Läser user_settings (cards, categories, bakgrundsfärg) · sparas debounced via useSettingsSync
AuthProvider
Supabase Auth · exponerar useAuth() → { user, loading }
DevStateProvider
?devState= query param · dev-only override av makro-mode
CoupleSpaceProvider
useCoupleSpace() · hämtar space · realtime på couple_spaces · auto-skapar utrymme vid första login · Vol 1: alltid single-member
NormalizedSessionProvider
useNormalizedSessionState() · anropar get_active_session_state RPC · realtime på couple_sessions + couple_session_completions
AppProvider
categories, cards, backgroundColor, savedConversations · innersta lagret · useSharedProgress är stub (no-op)
ProposalsProvider borttagen. Partner-toasts borttagna. Inget partnerlogik, ingen A/B-gating, ingen waiting-mode. Vol 1 = single writer, single space.
0B · Sessionsauktoritet — Single Writer Model

Dataflöde: Sessionsstart → Stegframsteg → Avslut

Laglås: All sessionslogik lever server-side.
Frontend läser passivt via NormalizedSessionContext. Ingen lokal sessionsflagga.
Förbjudet: couple_progress.current_session · journeyState · lokal manipulation · A/B-protokoll
activate-session (Edge Function)
Anropas vid samtalstart. Anropar activate_couple_session RPC. Returnerar session_id + steg.
activate_couple_session (RPC)
Skapar couple_sessions-rad + N couple_session_steps. Säkrar exakt en aktiv session per space.
complete_couple_session_step (RPC)
Markerar steg klart. Returnerar { is_step_complete, is_session_complete }. Ingen partner_left i Vol 1.
get_active_session_state (RPC)
Returnerar { mode, session_id, card_id, category_id, current_step_index, step_completions }. Anropas av NormalizedSessionProvider.
lock_step_reflections (RPC)
Låser step_reflections för session_id + step_index. Inga speaker-filter. Kallas vid stegövergång.
couple_sessions (tabell)
Auktoritativ. Status: active | completed. Realtime-prenumererad av NormalizedSessionProvider.
couple_session_completions (tabell)
En rad per steg när komplett. Realtime-prenumererad. Används av spaceSnapshot för progress.
step_reflections (tabell)
Unik per (session_id, step_index, user_id). State: draft → ready. Autosparas debounced 800ms.
couple_takeaways (tabell)
Gemensam sammanfattning per session. Autosparad 800ms. Låst vid session-exit.
Starta samtal → activate-session Edge Fn → activate_couple_session RPC → NormalizedSession refetch
Trycker "Fortsätt" → upsert step_reflections state:'ready' → complete_couple_session_step RPC → refetch → nästa steg
Sista steget klart → is_session_complete = true → Completion-mode monteras
0C · Makro-mode (useAppMode)

Tre möjliga makro-tillstånd

loading
normalizedSession.loading = true. UI visar pulse-loader. Ingen data renderas.
idle
sessionId = null. Hem visar kategorilista. Inga banners. Inga proposals.
active
sessionId + cardId finns. Hem visar resume-block. Kategorilistan döljs. ActiveSessionGuard blockerar navigering bort.
devState=pairedActive → mode: active. Alla övriga devState → mode: idle.
Borttaget i Vol 1: waiting-mode, proposalIncoming, partner_left, paired/duo-logik.
0D · CardView-lägesresolver

Fyra cardViewMode — en resolver avgör vilket som monteras

live
normalizedSession.sessionId finns + cardId matchar. SessionStepReflection monterat. Mutations aktiva.
revisit
?revisit=true i URL. Read-only. Inga session-mutations. SectionView utan reflektionsruta.
history
Avslutad session finns för kortet. CompletedSessionView monteras med reflektioner + takeaway.
completion
Session precis avslutad (is_session_complete = true). Takeaway-ritual monteras. Autospar debounced.
Resolver-ordning: isRevisitMode → hasActiveMatchingSession → hasCompletedSessionForCard → 'live' (default)

loading = true → Pulse loader (förhindrar flicker)

mode = idle → Fri navigation
Logotypen i Header leder alltid till /. Inga automatiska redirects vid session-events.

0G · Tillgänglighet & tap-targets

A11Y-regler (Feb 2026)

Listobjekt — min 44px höjd
Kategori- och ämnesrader har min-h-[44px] + flex flex-col justify-center. Hela raden är tryckmål (ej bara text).
CompletionMarker
SVG-ikonen har role="img" aria-label="Avklarad". Visuell storlek 12px men tryckmål utökat med 6px padding + negativa marginaler (≈24×24px). Påverkar inte radens tryckmål.
Listradar — aria-label
Interaktiva containrar i Category.tsx och Categories.tsx har aria-label som speglar korttiteln för skärmläsare.
Princip
Premium-känsla + tillgänglighet utan visuellt brus. Inga synliga etiketter på markeringen — enbart ARIA.
0H · Screenshot Export Tool

React-baserat skärmbildsverktyg — /screenshot-export

Inte public/screenshot-export.html — använd den inbyggda React-sidan /screenshot-export (kräver inloggning).
ScreenshotExport.tsx
Dashboard-sida. Startar loop via navigate() → steg 0. Visar grid när sc_done=1. ZIP-nedladdning via jszip.
useCaptureController
Kör inuti App på varje rutt. Detekterar ?__sc_step=N. Väntar 2500ms → html2canvas → sessionStorage → navigate nästa steg. Deduplicering via __sc_captured i sessionStorage (överlever re-mounts).
__sc_dev_step param
Tvingar CardView till ett visst stegindex utan RPC. Används för att fånga steg 1–3 i kortflödet.
