0B · Sessionsauktoritet — Single Writer Model (Volume 1)

Laglås: All sessionslogik och -auktoritet lever i servern.
Frontenden läser passivt och driver aldrig tillstånd lokalt.
Förbjudet: couple_progress.current_session · journeyState · lokal sessionsflagga · A/B-protokoll
activate_couple_session (RPC)
Startar ny session. Returnerar session_id. Anropas av Edge Function via proposal.
complete_couple_session_step (RPC)
Markerar ett steg klart. Returnerar {is_step_complete, is_session_complete, partner_left}.
get_active_session_state (RPC)
Hämtar {mode, session_id, card_id, category_id, current_step_index, step_completions}.
lock_step_reflections (RPC)
Låser alla reflektioner för session_id + step_index. Inga speaker-filter.
step_reflections (tabell)
Unik per (session_id, step_index, user_id). State: draft → ready. Ingen speaker_label i logiken.
couple_sessions (tabell)
Auktoritativ sessionstabell. Status: active | completed. Realtime-prenumererad.
couple_session_completions (tabell)
Per-user stegslutföranden. Realtime-prenumererad.
couple_takeaways (tabell)
Gemensam sammanfattning per session. Autosparad debounced 800ms. Låst vid session-exit.

0C · Dataflöde — Reflection save path

Användaren skriver → setText() → debounce 800ms → upsert step_reflections state:'draft'
Trycker "Fortsätt" → markReady() → upsert state:'ready' → complete_couple_session_step RPC → normalizedSession.refetch()
Session klar → is_session_complete = true → setShowCompletion(true) → CompletionMode monteras
onConflict: 'session_id,step_index,user_id' — garanterar exakt en rad per steg per användare.

0D · CardView-lägesresolvern

live
normalizedSession.sessionId finns + cardId matchar. SessionStepReflection monterat.
revisit
?revisit=true query param. Read-only. Ingen session-mutation.
history
Avslutad session finns för kortet. CompletedSessionView monteras.
completion
Session precis avslutad (showCompletion = true). Takeaway-ritual.
guard
Paired men ingen aktiv session. Blockerar — visas ej i Vol 1.
Resolver: isRevisitMode → hasActiveSession → hasCompletedSessionForCard → 'live' (default)

0E · Edge Functions

activate-session
Anropar activate_couple_session RPC. Returnerar session_id.
create-couple-space
Skapar nytt par-utrymme. Genererar invite_code + invite_token.
join-couple-space
Validerar token/code. Lägger till user i couple_members.
leave-and-create-new-space
Lämnar utrymme + skapar nytt för användaren.
switch-to-new-space
Byter till nytt utrymme utan partner.
send-notification-email
Skickar e-post vid proposal. 10-min deduplicering via system_events.
update-step-completion
Hjälpfunktion för stegslutföranden.
